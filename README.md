# SkillSphere – Smart Recruitment System

SkillSphere is a Django-based web recruitment platform developed for simplifying the hiring process for candidates, recruiters, and administrators.

## Features
- Smart skill-based job matching
- Candidate and recruiter modules
- Interview scheduling system
- Recruiter verification system
- Notifications and feedback
- Certificate upload and verification
- Django admin management

## Technologies Used
- Python Django
- HTML/CSS/Bootstrap
- JavaScript
- SQLite
